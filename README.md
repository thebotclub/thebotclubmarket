## Quick Stats (Projections)

- **Break-even:** Month 10 (~8,500 active users)
- **Year 1 Revenue:** ~$894K (GMV $5.96M)
- **Year 3 Revenue:** ~$27M (GMV $180M)
- **MVP Timeline:** 8-10 weeks to launch
- **Seed Capital Needed:** ~$150K (bootstrap to prove demand)

## Tech Stack
- Backend: Node.js/TypeScript (NestJS)
- Database: PostgreSQL + Redis
- Message Bus: NATS JetStream
- Object Storage: Cloudflare R2
- Payments: Stripe Connect + Credits system
- Infrastructure: Fly.io or Railway → AWS (at scale)
- Frontend: React + Tailwind
